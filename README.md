
自己鼓捣的一个基于springBoot的分布式 商城  （项目有很大的改进）

（我的服务器内存不足跑不起来那么多个服务，所以项目没有拆分成太多个服务，这有点违背分布式的概念了，不解耦）

### 技术：

 springboot

 ssm

mybatis

dubbo

zk

shiro

nginx

redis

消息中间件RabbitMq



## 包含前后台：

# 前台功能有：

  （前台用户nginx转发了两台服务）

1.浏览商品：

2.购买商品

3.加入购物车

4.申请退货

5.没有搜索功能（Es全文检索）（买的阿里云学生服务器，只有2G运行内存 跑不起来其它服务了，所以暂时先这样）

6.修改个人资料（头像上次）

7.收货地址管理

**8.有个秒杀抢购功能**

# **后台功能有**：

1.商品的管理

2.用户的管理

3.秒杀管理

4.角色管理

5.订单管理

6.在线用户管理

7.日志管理
